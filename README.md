# 🛒 Ecommerce - TalentoTech

Este proyecto consiste en el desarrollo de un sitio web de tienda ecommerce orientado a la venta de productos de computación.
El objetivo de la página es permitir a los usuarios navegar por el sitio y conocer los productos disponibles, contando con un diseño responsive que se adapta a distintos dispositivos.

## 🚀 Tecnologias

- HTML5
- CSS3

## 🤖 Uso de IA

Para el proyecto se hizo uso de la IA para la creacion de imagenes y textos de la seccion Testimonios

## ✨ Features

- Diseño responsive con Flexbox y Grid
- Navbar con Menu de hamburguesa para moviles
- Formulario integrado con Formspree  
- Seccion de Reseñas con animaciones en las tarjetas
- Grid de 3 columnas en reseñas y 4 columnas en productos
